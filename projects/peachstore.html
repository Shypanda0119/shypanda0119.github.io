🛒 Peach-Store – 전자기기 커머스

옵션 조합 로직 · 재고 차감 · 상품 상세

📌 프로젝트 개요

Peach-Store는 전자기기 옵션(기종/색상/용량)을 조합해 상품을 구성하고, 선택된 옵션에 따른 재고를 처리하는 커머스 프로젝트입니다.

🛠 기술 스택

  • Backend: Spring MVC
  • View: JSP + Bootstrap
  • Database: MySQL

🎯 담당 역할

1) 옵션 조합 로직

  • 기종/색상/용량 옵션 구조 설계
  • 옵션 조합 기반 상품 필터링

2) 재고 처리

  • 사용자 선택 → 해당 옵션 재고 차감 기능 구현
  • 재고 소진 시 처리

3) DB 설계

  • 상품/옵션/재고 테이블 구조 설계 참여
