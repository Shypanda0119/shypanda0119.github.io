🍽 Tabletopia – 식당 예약 & 관리자 플랫폼

관리자 페이지 백엔드 · AI 추천 · 지역 기반 식당 추천 · DB 설계 참여

📌 프로젝트 개요

Tabletopia는 전국 식당 정보를 기반으로 한 관리자 중심의 식당 관리·예약 플랫폼입니다. React 기반 관리자 페이지와 Spring Boot 기반 백엔드 서버로 구성되어 있으며, 관리자 CRUD, 지역 기반 추천, ChatGPT 기반 간단 추천 등 핵심 기능을 직접 구현했습니다.

🛠 기술 스택

  • Backend: Spring Boot, Spring MVC, Spring Data JPA
  • Database: MySQL
  • Tools: Docker, Git, GitHub Actions, Postman
  • AI: OpenAI API(ChatGPT 연동)

🎯 담당 역할

1) 관리자 페이지 백엔드 CRUD 전체 개발

  • 식당 CRUD
  • 메뉴 CRUD
  • 운영시간 CRUD
  • 편의시설 CRUD
  • 이미지 업로드 API
  • 리뷰(댓글) CRUD
  • React Admin과 API 연동

2) 지역 기반 추천 로직 개발

  • 주소 문자열에서 “구 단위” 추출
  • 지역 필터링으로 후보 식당 리스트 구성
  • DB 기반 지역별 식당 추천 데이터 생성

3) ChatGPT 기반 추천 기능 개발

  • OpenAI API 연동
  • 프롬프트 구성
  • 응답 파싱 로직 개발
  • 간단한 5턴 내외 대화형 추천 기능

4) JPA 사용 경험

  • 쿼리 최적화 필요 시 fetch join 적용 경험
  • DTO 기반 조회 방식 적용
  • JPQL보다는 Spring Data JPA 중심 개발

5) DB 설계 참여

  • 식당/메뉴/운영시간/이미지/리뷰 테이블 구조
  • 관계(FK) 및 기본 인덱스 설계
