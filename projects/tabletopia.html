🍽 Tabletopia — 식당 예약 & 관리 플랫폼

Spring Boot JPA MySQL OpenAI API React Admin

📌 역할 (My Role)

관리자 백엔드 개발 · 지역 판별 로직 개발 · AI 추천 기능 · DB 설계 참여

📌 주요 구현 기능

  • 관리자 백엔드 CRUD
    • 식당, 메뉴, 운영시간, 이미지, 편의시설, 리뷰(댓글형식) API
    • React Admin과 연동되는 데이터 구조 설계
  • 지역 기반 추천 로직
    • 주소 문자열에서 "구" 단위 추출 로직 구현
    • 지역별 식당 필터링 알고리즘 작성
  • ChatGPT 기반 추천 기능
    • OpenAI API 요청/응답 파싱
    • 5회 제한 간단 대화형 추천 기능
  • JPA 사용
    • fetch join 적용 경험(N+1 해결)
    • DTO 기반 조회 중심
  • DB 설계
    • 식당/메뉴/운영시간/이미지/리뷰 테이블 설계

📌 결과

관리자 페이지와 실시간 추천 기능이 완성되며 서비스 구조가 명확해졌고, 실제 식당 관리/추천 시나리오를 백엔드에서 완전하게 동작시킴.
