사진 업로드와 댓글 기능을 제공하는 미니 SNS 서비스 프로젝트입니다. Firebase Auth를 활용해 인증 기능을 구현하고, 게시물과 댓글 흐름을 중심으로 SNS의 기본 기능을 구현했습니다.

Spring Boot Firebase Auth Airtable

프로젝트 소개

  • 사진 업로드와 댓글 기능을 제공하는 미니 SNS 서비스 구현
  • Firebase Auth 기반 로그인 / 회원가입 기능 연동
  • 게시물과 댓글 흐름을 중심으로 SNS 기본 기능 구현

담당 역할

  • Firebase 인증 연동
  • 게시물 CRUD 및 이미지 업로드 기능 구현
  • 댓글 CRUD 전체 기능 구현
  • 사용자별 게시물 구분 로직 구현

핵심 구현 기능

인증 기능 구현

  • Firebase Auth 기반 로그인 / 회원가입 기능 연동
  • 인증 기능을 빠르게 구현하기 위해 Firebase Auth를 사용
  • 직접 JWT 인증 구조를 구현하는 대신 외부 인증 서비스를 연동해 핵심 기능 개발에 집중

게시물 기능 구현

  • 전체 게시물 조회 기능 구현
  • 본인 게시물만 조회할 수 있는 사용자별 구분 로직 구현
  • 이미지 업로드와 게시물 등록 기능 연결

댓글 기능 구현

  • 댓글 등록 / 조회 / 수정 / 삭제 기능 구현
  • 게시물 상세 화면에서 댓글 목록 조회 가능하도록 구성
  • 게시물과 댓글 간 연결 관계를 고려한 데이터 흐름 설계

기술적 문제와 해결

이미지 업로드 시 용량 제한 문제

  • 업로드 가능한 이미지 용량을 초과하는 파일이 들어오는 문제 발생
  • 업로드 파일 크기 제한 필터링을 적용해 예외 상황 처리

Airtable API 필드 제한 문제

  • Airtable API 사용 과정에서 필드 수 제한으로 데이터 관리 제약 발생
  • 필드를 분리하여 데이터 구조를 조정하고 제약 해결

결과

  • Firebase 인증 기반 SNS 기본 기능 구현
  • 게시물과 댓글 흐름을 연결한 서비스 구조 완성
  • 외부 인증 및 데이터 서비스 연동 경험 확보
