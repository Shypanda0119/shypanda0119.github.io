📦 Shinlogis-WMS – 창고 관리 시스템

재고 흐름 처리 · 유통기한/파손 로직 · 대시보드 데이터 API

📌 프로젝트 개요

Shinlogis-WMS는 실제 창고 운영 방식을 기반으로 하는 재고 관리 시스템입니다. 재고 상태(정상/임박/파손) 분류 및 입·출고 흐름을 백엔드에서 처리하며 Swing 기반 UI에서 사용할 데이터를 제공했습니다.

🛠 기술 스택

  • Backend: Spring Boot, JPA
  • Database: MySQL
  • Tools: Git, Postman
  • UI: Java Swing (백엔드 연동)

🎯 담당 역할

1) 재고 흐름 처리

  • 유통기한 임박/파손 자동 분류
  • 출고·폐기 요청 시 재고 차감 로직 개발

2) 대시보드용 API

  • 입·출고 수량 합계 제공
  • 현재 재고 상태 요약 제공
  • Swing UI 대시보드 데이터 제공

3) DB 설계 참여

  • 재고/입고/출고/폐기 테이블 설계
  • 상태(enum) 구조 설계
