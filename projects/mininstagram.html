📸 MiniInstagram — 미니 SNS 서비스

Spring Boot Firebase Auth Airtable

📌 프로젝트 개요

사용자가 사진을 올리고 댓글을 남길 수 있는 간단한 SNS 서비스입니다. JWT 인증 대신 Firebase 인증을 사용하여 로그인/회원 관리를 구현했습니다.

📌 담당 역할 (My Role)

  • Firebase 인증 연동
  • 게시물 CRUD + 이미지 업로드
  • 댓글 CRUD 전체 기능 구현
  • 사용자별 게시물 구분 로직

📌 구현 기능 상세

① 인증

  • Firebase Auth 기반 로그인/회원가입
  • JWT·Spring Security 사용 없음

② 게시물 기능

  • 전체 게시물 조회
  • 본인 게시물만 조회하는 기능
  • 이미지 업로드 + 게시물 등록

③ 댓글 기능

  • 댓글 등록/조회/수정/삭제 기능 구현
  • 게시물 상세화면에서 댓글 목록 제공

📌 기술적 문제 & 해결

  • 이미지 업로드 시 용량 제한 → 용량 필터링 추가
  • Airtable API의 필드 제한 → 컬럼 분리하여 해결

📌 결과

인증 기반 SNS 기본 기능을 구축하며 API 설계와 데이터 흐름에 대한 경험을 쌓았습니다.
