백엔드 개발자 김지민

데이터 흐름과 운영 로직을 설계하는 백엔드 개발자입니다.

Spring Boot와 MySQL 기반으로 백엔드 서비스를 구현하며
예약, 재고, 추천, 옵션 조합처럼 상태 변화가 많은 기능을 개발했습니다.
단순한 CRUD 구현을 넘어 데이터 구조와 API 흐름이 실제로 안정적으로 동작하도록 설계하는 것을 중요하게 생각합니다.

Core Strengths

API 설계 및 구현

Spring Boot 기반 REST API를 설계하고
관리자 기능과 사용자 기능을 프로젝트에 구현했습니다.

데이터 구조 설계

MySQL 테이블 설계와 조회 로직 구성을 통해
서비스 흐름에 맞는 데이터 구조를 설계했습니다.

운영 로직 처리

트랜잭션, 예외 처리, Soft Delete 등을 적용해
데이터 일관성과 운영 안정성을 고려한 로직을 구현했습니다.
